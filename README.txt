TOPDJS CRM v11.4.24 - CALENDARIO ANUAL CON AÑOS

Corrección:
- Se mantiene el calendario simplificado sin selector de mes/año ni botón IR.
- Se agregan botones para navegar años anteriores y posteriores.
- Controles visibles: HOY, ← AÑO, VER AÑO COMPLETO, AÑO →.
- En vista anual, ← AÑO y AÑO → cambian el año completo mostrado.
- Al seleccionar un mes de la vista anual, se abre ese mes con sus fechas y eventos.

Instalación:
1. Subir los 7 archivos a GitHub.
2. Commit: TopDJs CRM v11.4.24 - Calendario anual con años
3. Deploy to Production en Vercel.
4. Recarga fuerte: Option + Command + R.
